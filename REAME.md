## Cài đặt
1. Clone repository

2. Tạo môi trường ảo (Virtual Environment)
```python
# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python3 -m venv venv
source venv/bin/activate
```
3. Cài đặt các thư viện cần thiết
```python
pip install -r requirements.txt
```

## Cách chạy ứng dụng
Chạy lệnh:
```
python app.py
```

Ứng dụng sẽ chạy trên http://127.0.0.1:5000/.
